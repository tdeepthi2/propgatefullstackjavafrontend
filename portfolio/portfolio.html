About Portfolio Entity

The Portfolio entity acts as a consolidated performance tracking record for property holdings. It gathers key summary metrics like total deployed capital, cumulative rental incomes, overall Return on Investment (ROI), and active health ratings.

Portfolio Indicators:

Four key indicators outline portfolio performance. Total Investment sums up the original acquisition costs for all properties in the portfolio. Total Income represents the aggregated cash returns generated via lease payouts and capital appreciations. ROI (%) measures the net profit efficiency percentage relative to the original cost basis. The Health Score is a composite metric rating (0-100) representing portfolio stability, tenant occupancy, and debt-to-equity ratios.

Portfolio Return & Health Calculations

Portfolios aggregate individual property achievements to offer investors a single unified high-level performance assessment. The Aggregate ROI is calculated as ((Current Property Values + Cumulative Rental Income) - Total Deployed Capital) / Total Deployed Capital * 100 to determine real net wealth generation. The Health Score Engine computes a value dynamically from 0 to 100 based on a weighted combination of average vacancy rate, rent collection compliance, and property status distribution. Through Data Syncing, a portfolio is updated in real-time or via scheduled tasks whenever new property purchases, rental payouts, or appreciation schedules are registered.
